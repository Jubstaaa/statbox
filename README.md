# StatBox

League of Legends widget builder for OBS.

## Setup

1. Install dependencies:

```bash
bun install
```

2. Create your env file:

```bash
cp .env.example .env.local
```

3. Add your Riot API key to `.env.local`.

4. Start the app:

```bash
bun run dev
```

## Scripts

```bash
bun run dev
bun run lint
bun run build
```
